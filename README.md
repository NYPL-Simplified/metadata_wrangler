Simplified-server
=================

Library Simplified Server

sudo apt-get install python-virtualenv python-dev libpq-dev libxml2-dev libxslt-dev
virtualenv venv
. venv/bin/activate

pip install pyatom
pip install lxml
pip install beautifulsoup4
pip install requests
pip install rdflib
pip install gunicorn
pip install isbnlib
pip install pillow

pip install pyld
